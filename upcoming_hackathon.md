# Overview

Seoul AI is hosting fourth AI hackathon on Saturday, December 15th. Hackathon is based on the toolkit from Seoul AI for developing AI algorithms: Seoul AI Gym. This gym simulates various environments and enables the use of any teaching technique on an agent. The goal of every hackathon participant is to develop an agent that can do Cryptocurrency trading. Each participant will train an agent to compete for the first place. We encourage participants to use any machine learning algorithm of their choice. Dataset will be given. Each participant will have their own responsibility to find a way to train their own agent.
